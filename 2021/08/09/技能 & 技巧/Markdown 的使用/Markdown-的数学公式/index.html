LaTeX 的常用数学符号

数学运算符号

  • 点乘, 如 $a \cdot b$
  • 叉乘, 如 $a \times b$
  • 除以, 如 $a \div b$
  • 求和, 如 $\sum_{a}^{b}{c}$
  • 积分, 如 $\int_{a}^{b}{c}$
  • 累乘, 如 $\prod_{a}^{b}{c}$

  • $\bigcup_{a \in A}$

  • 分数, 如 $\frac{a}{b}$

  • 行间分式, 如 $\displaystyle \frac{a}{b}$
  • 偏导数符号, 如 $\partial$
  • 梯度, 如 $\nabla$
  • 根号, 如 $\sqrt{x}$
  • $a \odot b$
  • $a \oplus b$
  • $a \otimes b$
  • $a \cong b$
  • $a \propto b$
  • $\pm$


集合的相关操作

  • 真包含, 如 $a \subset b$
  • 包含, 如 $a \subseteq b$
  • 如 $a \supset b$
  • 如 $a \supseteq b$
  • 属于, 如 $a \in b$
  • 交集, 如 $a \cap b$
  • 并集, 如 $a \cup b$
  • 不属于, 如 $a \notin b$
  • 空集, 如 $\emptyset$
  • 更好看的空集, 如 $\varnothing$


数理逻辑的相关操作

  • 任意, 如 $\forall$
  • 存在, 如 $\exists$
  • 或, 如 $\vee$ 也可以使用以下:
  • 与, 如 $\wedge$ 也可以使用以下:
  • 蕴含, 如 $\rightarrow$
  • 等价, 如 $\leftrightarrow$
  • 非, 如 $\neg$ 或
  • 满足, 如 $\models$
  • 推理, 如 $\vdash$
  • 反方向的推理符号, 如 $\dashv$
  • 定义为, 如 $\triangleq$


概率论相关符号

  • 独立, 如 $\perp !!! \perp$
  • 不独立, 如 $\not ! \perp !!! \perp$

数学关系符号

  • 恒等于, 如 $a \equiv b$
  • 大于等于, 如 $a \geq b$
  • 小于等于, 如 $a \leq b$
  • 不等于, 如 $a \neq b$
  • 相似于, 如 $a \sim b$
  • 两条竖线, 如 $a \parallel b$
    (暂时不知道代表什么数学概念)
  • 约等于, 如 $a \approx b$
  • 另外一种约等于, 如 $a \simeq b$


希腊字母

  • $\phi$
  • $\mu$
  • $\gamma$
  • $\varepsilon$
  • $\varphi$
  • $\lambda$
  • $\sigma$
  • $\zeta$
  • $\wp$
  • $\chi$
  • $\eta$
  • $\xi$

省略号

  • 横向省略号, 如 $\cdots$
  • 竖向省略号, 如 $\vdots$
  • 对角线方向的省略号, 如 $\ddots$


粗体和带箭头的向量

  • 粗体 1, 如 $\boldsymbol{x}$
  • 粗体 2, 如 $\mathbf{x}$
  • 带箭头的向量, 如 $\vec{x}$
  • 上_左箭头, 如 $\overleftarrow{x}$
  • 上_右箭头, 如 $\overrightarrow{x}$
  • 下_左箭头, 如 $\underleftarrow{x}$
  • 下_右箭头, 如 $\underrightarrow{x}$
  • $\Rightarrow$、$\Leftarrow$


空格

暂不清楚什么时间应用什么空格
  • 空格距离：3/18 em, 如 $a \, b$
  • 空格距离：4/18 em, 如 $a \: b$
  • 空格距离：5/18 em, 如 $a \; b$
  • 空格距离：1 em, 如 $a \quad b$
  • 空格距离：2 em, 如 $a \qquad b$

注：em 是指当前文本的字体尺寸

其他

LaTex 把下标放在正下方

  • 如果是数学符号, 那么直接在 _ 前加 \limits 命令, 即可把符号放在正下方, 如 $\sum \limits_{a}^{b}{c}$
  • 如果是普通符号, 那么要用 \mathop 先转成数学符号再用 \limits, 如 $\mathop{max} \limits_{a, b}$
  • 对于双美元符之间的公式, LaTeX 默认下标放在正下方


LaTex 在箭头上写字

  • 如 $a \overset{c}{\longrightarrow} b$


LaTex 字符上面加符号

  • 加 ^, 如 $\hat{a}$ 或 $\widehat{a}$
    $\hat{a}$ $\widehat{a}$
  • 加一个点, $\dot{x}$
  • 加两个点, $\ddot{x}$
  • 加横线, 如 $\overline{a}$
  • 加波浪线, 如 $\widetilde{a}$
  • 加一个点, 如 $\dot{a}$
  • 加两个点, 如 $\ddot{a}$
